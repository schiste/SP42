# Code of Conduct

SP42 is an open-source project for Wikimedia patrol tooling. Participation is
open to contributors who work respectfully, protect user safety, and keep
project discussions focused on improving the software.

## Expected Behavior

- Be direct, constructive, and respectful.
- Assume good faith, but respond to impact when something causes harm.
- Keep technical disagreement about the work, not the person.
- Respect maintainer decisions about project scope, security, and deployment.
- Avoid sharing private credentials, private user data, or sensitive security
  details in public channels.

## Unacceptable Behavior

- Harassment, threats, insults, or sustained personal attacks.
- Discriminatory language or exclusionary behavior.
- Publishing private information without explicit permission.
- Sexualized, violent, or otherwise hostile conduct.
- Public disclosure of vulnerabilities before maintainers have had a reasonable
  chance to triage and respond.
- Deliberately disruptive behavior in issues, pull requests, reviews, or chat.

## Enforcement

Maintainers may moderate, edit, hide, or remove comments and other contributions
that violate this code. They may also close issues, reject pull requests, or
limit participation when needed to protect contributors and project users.

For conduct concerns, contact a project maintainer privately. For security
vulnerabilities, follow [SECURITY.md](SECURITY.md).
